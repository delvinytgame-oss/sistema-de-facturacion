# Sistema de Facturación

He creado la estructura de carpetas necesaria para este proyecto manualmente dado que Node.js y Python no están instalados.

## 1. Frontend (React + Tailwind)
La carpeta `frontend` contiene los archivos de código fuente de React usando Vite.

**Una vez que instales Node.js (https://nodejs.org/):**
1. Abre tu terminal.
2. Entra a la carpeta: `cd "c:\Users\Deibinson\Desktop\sistena de facturacion\frontend"`
3. Instala dependencias: `npm install`
4. Inicia el servidor de desarrollo: `npm run dev`

## 2. Backend (Flask + PostgreSQL)
La carpeta `backend` contiene el código base para la API con Flask.

**Una vez que instales Python (https://www.python.org/downloads/):**
1. Abre tu terminal.
2. Entra a la carpeta: `cd "c:\Users\Deibinson\Desktop\sistena de facturacion\backend"`
3. Crea un entorno virtual: `python -m venv venv` (o usa `py -m venv venv`)
4. Activa el entorno: `.\venv\Scripts\activate`
5. Instala paquetes: `pip install -r requirements.txt`
6. Corre el servidor: `python app.py`

*Nota: No olvides instalar Node.js y Python en tu computadora, la opción "Add Python to PATH" debe estar marcada al instalar.*
